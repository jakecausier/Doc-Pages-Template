Getting Started

Installation

To install the framework, navigate into your website's Laravel Installation containing your composer.json within your Terminal window.

Laravel E-commerce required that the Auth module is enabled and imported to your Laravel project. Import it using php artisan make:auth

Ensure you have Composer installed, and run composer install chrisbraybrooke/laravel-ecommerce. This will install the latest version of the package into your composer.json .

After installation, you will have access to the Artisan command ecommerce, where you can begin to implement the PurpleCommerce system.

To install the system into your Laravel configuration, run php artisan ecommerce:install and follow the instructions.
